# Excel Dashboard 建立指南

## Dashboard.xlsx 結構

本文件說明如何建立 Dashboard.xlsx，因為無法直接生成 Excel 檔案，請依照以下步驟手動建立。

---

## Sheet 1: 專案總覽 (Project Overview)

### 基本資訊區 (A1:G10)

```
A1: 專案名稱          B1: I&C Project Management
A2: 專案經理          B2: [您的名字]
A3: 開始日期          B3: 2025-01-01
A4: 預計結束日期      B4: 2025-06-30
A5: 實際結束日期      B5: [空白]
A6: 專案狀態          B6: 進行中
A7: 整體進度          B7: =COUNTIF(任務清單!F:F,"已完成")/COUNTA(任務清單!F:F)*100 & "%"
```

### KPI 卡片區 (A12:D20)

#### 卡片 1: 總任務數 (A12:B15)
```
A12: 總任務數
A13: =COUNTA(任務清單!B2:B100)
格式: 字體 48pt, 粗體, 藍色
```

#### 卡片 2: 已完成 (C12:D15)
```
C12: 已完成
C13: =COUNTIF(任務清單!F:F,"已完成")
格式: 字體 48pt, 粗體, 綠色
```

#### 卡片 3: 進行中 (A17:B20)
```
A17: 進行中
A18: =COUNTIF(任務清單!F:F,"進行中")
格式: 字體 48pt, 粗體, 橙色
```

#### 卡片 4: 逾期任務 (C17:D20)
```
C17: 逾期任務
C18: =COUNTIFS(任務清單!E:E,"<"&TODAY(),任務清單!F:F,"<>已完成")
格式: 字體 48pt, 粗體, 紅色
```

### 圖表區

#### 圖表 1: 任務狀態分布 (F12:J20)
- 類型: 圓餅圖 (Pie Chart)
- 資料來源: 統計分析!A2:B5
- 標題: "任務狀態分布"

#### 圖表 2: 優先級分布 (F22:J30)
- 類型: 環形圖 (Donut Chart)
- 資料來源: 統計分析!D2:E4
- 標題: "優先級分布"

#### 圖表 3: 團隊成員工作量 (A22:E35)
- 類型: 橫條圖 (Bar Chart)
- 資料來源: 統計分析!G2:H10
- 標題: "團隊成員工作量"

#### 圖表 4: 完成趨勢 (F32:J45)
- 類型: 折線圖 (Line Chart)
- 資料來源: 統計分析!J2:K20
- 標題: "每週完成任務趨勢"

## Sheet 2: 任務清單 (Task List)

### 表頭 (Row 1)

```
A1: 任務ID
B1: 任務名稱
C1: 負責人
D1: 優先級
E1: 截止日期
F1: 狀態
G1: 完成日期
H1: 進度%
I1: 類別
J1: 備註
```

### 資料驗證設定

#### C 欄 (負責人)
- 資料驗證 > 清單
- 來源: 張三,李四,王五,趙六,專案經理

#### D 欄 (優先級)
- 資料驗證 > 清單
- 來源: 🔴高,🟡中,🟢低

#### F 欄 (狀態)
- 資料驗證 > 清單
- 來源: 待辦,進行中,審核中,已完成

#### I 欄 (類別)
- 資料驗證 > 清單
- 來源: 管理,技術,文件

### 條件格式設定

#### D 欄 (優先級)
1. 規則: =$D2="🔴高"
   - 格式: 紅色背景 (#FF0000), 白色文字

2. 規則: =$D2="🟡中"
   - 格式: 黃色背景 (#FFFF00), 黑色文字

3. 規則: =$D2="🟢低"
   - 格式: 綠色背景 (#00FF00), 黑色文字

#### F 欄 (狀態)
1. 規則: =$F2="待辦"
   - 格式: 灰色背景 (#CCCCCC)

2. 規則: =$F2="進行中"
   - 格式: 藍色背景 (#0066CC), 白色文字

3. 規則: =$F2="審核中"
   - 格式: 橙色背景 (#FF9900), 白色文字

4. 規則: =$F2="已完成"
   - 格式: 綠色背景 (#00CC00), 白色文字

#### E 欄 (截止日期)
1. 規則: =AND($E2<TODAY(),$F2<>"已完成")
   - 格式: 紅色粗體

2. 規則: =AND($E2<=TODAY()+3,$E2>=TODAY(),$F2<>"已完成")
   - 格式: 黃色背景

#### H 欄 (進度%)
- 資料橫條 (Data Bar)
- 顏色: 藍色漸層

## Sheet 3: 甘特圖 (Gantt Chart)

### 設計說明

甘特圖使用條件格式建立，不需要額外圖表。

### 表頭設定

```
A1: 任務ID
B1: 任務名稱
C1: 負責人
D1: 開始日期
E1: 結束日期
F1: 進度%
G1: 2025/1/1  (日期，向右延伸至專案結束日)
H1: 2025/1/2
I1: 2025/1/3
... (每欄代表一天)
```

### 資料區 (從 Row 2 開始)

```
A2: =任務清單!A2
B2: =任務清單!B2
C2: =任務清單!C2
D2: =任務清單!E2-7  (假設任務期間 7 天)
E2: =任務清單!E2
F2: =任務清單!H2
```

### 條件格式 (套用至 G2:ZZ100)

#### 規則 1: 任務期間 (藍色)
```
公式: =AND($D2<=G$1,G$1<=$E2)
格式: 藍色背景 (#0066CC)
```

#### 規則 2: 已完成進度 (深藍色)
```
公式: =AND($D2<=G$1,G$1<=$D2+($E2-$D2)*$F2/100)
格式: 深藍色背景 (#003366)
```

#### 規則 3: 今日線 (紅色)
```
公式: =G$1=TODAY()
格式: 紅色背景 (#FF0000)
```

### 格式設定

- 欄寬: G 欄之後所有欄位設為 3
- 列高: 20
- 日期格式: G1 開始的日期欄位格式為 "M/D"

## Sheet 4: 週報 (Weekly Report)

### 週資訊區 (A1:B5)

```
A1: 週報期間
B1: =TEXT(TODAY()-WEEKDAY(TODAY())+1,"yyyy/mm/dd") & " - " & TEXT(TODAY()-WEEKDAY(TODAY())+7,"yyyy/mm/dd")

A2: 報告日期
B2: =TODAY()

A3: 報告人
B3: [自動從專案總覽!B2 取得]
```

### KPI 區 (A7:D15)

```
A7: 本週完成任務數
B7: =COUNTIFS(任務清單!G:G,">="&TODAY()-7,任務清單!G:G,"<="&TODAY(),任務清單!F:F,"已完成")

A8: 本週新增任務數
B8: [手動輸入或從歷史記錄計算]

A9: 本週逾期任務數
C9: =COUNTIFS(任務清單!E:E,"<"&TODAY(),任務清單!F:F,"<>已完成")

A10: 整體進度變化
B10: [本週進度% - 上週進度%]
```

### 圖表 1: 任務完成趨勢 (A17:F30)

- 類型: 折線圖
- X 軸: 週次 (Week 1, Week 2, ...)
- Y 軸: 完成任務數
- 資料來源: 手動維護的週次統計表

### 圖表 2: 負責人工作量 (H17:M30)

- 類型: 堆疊橫條圖
- 資料來源: 統計分析!G2:I10
- 系列: 待辦、進行中、已完成

### 本週重點 (A32:M40)

```
A32: 本週成就
A33: [手動輸入]

A35: 本週挑戰
A36: [手動輸入]

A38: 下週計畫
A39: [手動輸入]
```

### 逾期任務清單 (A42:J50)

```
A42: 逾期任務清單
表頭: 任務ID | 任務名稱 | 負責人 | 截止日期 | 逾期天數 | 狀態 | 行動計畫

公式範例:
E43: =TODAY()-[截止日期]
```

## Sheet 5: 統計分析 (Analytics)

### 任務狀態統計表 (A1:B5)

```
A1: 狀態        B1: 數量
A2: 待辦        B2: =COUNTIF(任務清單!F:F,"待辦")
A3: 進行中      B3: =COUNTIF(任務清單!F:F,"進行中")
A4: 審核中      B4: =COUNTIF(任務清單!F:F,"審核中")
A5: 已完成      B5: =COUNTIF(任務清單!F:F,"已完成")
```

### 優先級統計表 (D1:E4)

```
D1: 優先級      E1: 數量
D2: 🔴高        E2: =COUNTIF(任務清單!D:D,"🔴高")
D3: 🟡中        E3: =COUNTIF(任務清單!D:D,"🟡中")
D4: 🟢低        E4: =COUNTIF(任務清單!D:D,"🟢低")
```

### 負責人工作量統計表 (G1:I10)

```
G1: 負責人      H1: 任務數      I1: 已完成數
G2: 張三        H2: =COUNTIF(任務清單!C:C,"張三")        I2: =COUNTIFS(任務清單!C:C,"張三",任務清單!F:F,"已完成")
G3: 李四        H3: =COUNTIF(任務清單!C:C,"李四")        I3: =COUNTIFS(任務清單!C:C,"李四",任務清單!F:F,"已完成")
G4: 王五        H4: =COUNTIF(任務清單!C:C,"王五")        I4: =COUNTIFS(任務清單!C:C,"王五",任務清單!F:F,"已完成")
G5: 趙六        H5: =COUNTIF(任務清單!C:C,"趙六")        I5: =COUNTIFS(任務清單!C:C,"趙六",任務清單!F:F,"已完成")
G6: 專案經理    H6: =COUNTIF(任務清單!C:C,"專案經理")    I6: =COUNTIFS(任務清單!C:C,"專案經理",任務清單!F:F,"已完成")
```

### 每週完成趨勢表 (J1:K20)

```

J2: Week 1      K2: [手動輸入或公式計算]

...
```

## 進階功能：樞紐分析表

- 資料來源: 任務清單!A:J
- 列: 負責人
- 值: 計數 - 任務ID, 平均 - 進度%
- 篩選: 狀態

### 樞紐分析表 2: 依狀態統計

- 資料來源: 任務清單!A:J
- 列: 狀態
- 欄: 優先級
- 值: 計數 - 任務ID

## 快速建立步驟

### 步驟 1: 建立檔案
1. 開啟 Excel
2. 另存新檔為 "Dashboard.xlsx"

### 步驟 2: 建立 Sheets
1. 重新命名 Sheet1 為 "專案總覽"
2. 新增 Sheet: "任務清單"
3. 新增 Sheet: "甘特圖"
4. 新增 Sheet: "週報"
5. 新增 Sheet: "統計分析"

### 步驟 3: 依照上述說明填入內容
- 從「任務清單」開始（這是資料基礎）
- 再建立「統計分析」（提供圖表資料來源）
- 然後建立「專案總覽」（使用統計分析的圖表）
- 接著建立「甘特圖」
- 最後建立「週報」

### 步驟 4: 美化格式
- 使用表格樣式
- 設定凍結窗格
- 調整欄寬列高
- 套用主題色彩

### 步驟 5: 測試公式
- 確認所有公式正確運作
- 測試條件格式
- 驗證資料驗證

---

## 注意事項

1. **日期格式**: 確保所有日期欄位格式一致
2. **公式參照**: 使用絕對參照 ($) 避免公式錯誤
3. **資料驗證**: 設定下拉選單確保資料一致性
4. **條件格式**: 注意套用範圍，避免影響效能
5. **圖表更新**: 圖表會自動更新，但需確保資料來源範圍足夠大

---

## 匯出與共享

### 匯出 PDF
1. 檔案 > 匯出 > 建立 PDF/XPS
2. 選擇要匯出的 Sheets
3. 設定頁面配置

### 上傳至 OneDrive
1. 儲存至 OneDrive
2. 設定共享權限
3. 產生共享連結

### 整合至 Teams
1. 在 Teams 頻道新增 Excel 分頁
2. 選擇 OneDrive 中的 Dashboard.xlsx
3. 團隊成員可即時協作
