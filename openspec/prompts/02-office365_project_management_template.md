## Request

設計一個微軟 Office 365 專案管理方案模板（包含範例），整合以下工具：

- Microsoft Planner
- Microsoft To Do
- Microsoft OneNote
- Microsoft Excel
- Microsoft Teams
- Microsoft Outlook

## 5 個核心流程

### 1. 專案啟動 (Project Initiation)

- 建立專案目標與範圍
- 定義利害關係人
- 設定專案里程碑
- 建立初始任務清單

### 2. 任務分配 (Task Assignment)

- 拆解專案任務
- 分配負責人
- 設定截止日期
- 建立子任務

### 3. 日常追蹤 (Daily Tracking)

- 更新任務狀態
- 記錄每日進度
- 同步會議紀錄
- 處理即時問題

### 4. 週檢討 (Weekly Review)

- 檢視週進度
- 分析任務瓶頸
- 調整優先順序
- 更新甘特圖

### 5. 結案總報告 (Project Closure)

- 完成度統計
- 成果交付
- 經驗總結
- 文件歸檔
