## 階段 3：Excel Dashboard 建立 (60 分鐘)

### 3.1 建立 Dashboard.xlsx

#### Sheet 1: 專案總覽 (Project Overview)

**欄位設計**：

| 欄位 | 說明         |
| ---- | ------------ |
| A    | 專案名稱     |
| B    | 專案經理     |
| C    | 開始日期     |
| D    | 預計結束日期 |
| E    | 實際結束日期 |
| F    | 專案狀態     |
| G    | 整體進度 %   |

**視覺化元素**：

- [ ] 專案進度環形圖 (Donut Chart)
- [ ] 任務狀態分布圓餅圖 (Pie Chart)
- [ ] 本週完成任務數 (Card)
- [ ] 逾期任務數 (Card with conditional formatting)
- [ ] 團隊成員工作量分布 (Bar Chart)

**公式範例**：

```excel
// 整體進度計算
=COUNTIF(任務清單!F:F,"已完成")/COUNTA(任務清單!F:F)*100

// 逾期任務計算
=COUNTIFS(任務清單!E:E,"<"&TODAY(),任務清單!F:F,"<>已完成")

// 本週完成任務
=COUNTIFS(任務清單!G:G,">="&TODAY()-7,任務清單!F:F,"已完成")
```

#### Sheet 2: 任務清單 (Task List)

**欄位設計**：

| 欄  | 欄位名稱 | 說明                      | 資料驗證 |
| --- | -------- | ------------------------- | -------- |
| A   | 任務 ID  | 自動編號                  | -        |
| B   | 任務名稱 | 任務描述                  | -        |
| C   | 負責人   | 指派對象                  | 下拉選單 |
| D   | 優先級   | 🔴高/🟡中/🟢低               | 下拉選單 |
| E   | 截止日期 | 完成期限                  | 日期格式 |
| F   | 狀態     | 待辦/進行中/審核中/已完成 | 下拉選單 |
| G   | 完成日期 | 實際完成日                | 日期格式 |
| H   | 進度 %   | 0-100%                    | 數字     |
| I   | 類別     | 管理/技術/文件            | 下拉選單 |
| J   | 備註     | 額外說明                  | -        |

**條件格式設定**：

- [ ] 優先級：🔴 紅色背景、🟡 黃色背景、🟢 綠色背景
- [ ] 狀態：
  - 待辦：灰色
  - 進行中：藍色
  - 審核中：橙色
  - 已完成：綠色
- [ ] 截止日期：
  - 已逾期：紅色粗體
  - 3 天內到期：黃色背景
  - 未來日期：正常
- [ ] 進度：資料橫條 (Data Bar)

#### Sheet 3: 甘特圖 (Gantt Chart)

**設計方式**：

- [ ] 使用條件格式建立視覺化甘特圖
- [ ] X 軸：日期（週為單位）
- [ ] Y 軸：任務名稱
- [ ] 顯示：開始日期、結束日期、進度

**公式範例**：

```excel
// 甘特圖條件格式公式
=AND($D2<=E$1,E$1<=$E2)

// 進度顯示
=AND($D2<=E$1,E$1<=$D2+($E2-$D2)*$H2/100)
```

**視覺化**：

- [ ] 任務條：藍色
- [ ] 已完成部分：深藍色
- [ ] 里程碑：菱形標記
